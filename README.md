# arhi-post-editor

Telegram WebApp — визуальный редактор rich-постов для @arhiGC_bot (Arhischool).
Пишет Markdown, отдаёт его боту через `Telegram.WebApp.sendData()` → бот показывает
rich-превью и публикует по кнопке. Публикация заперта в боте (аллоулист tg-id).

Хостинг: GitHub Pages. Источник правды страницы — `index.html`.
